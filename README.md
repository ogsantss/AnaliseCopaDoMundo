# ⚽ Análise Copa do Mundo 2026

Análise exploratória das 48 seleções classificadas para a Copa do Mundo 2026, utilizando dados públicos do Kaggle.

## 📊 O que foi feito

- Download e carregamento do dataset via KaggleHub
- Tradução de todas as colunas para português
- Tratamento de dados — nulos, duplicatas, tipos e consistência
- Análise Exploratória (EDA) com 5 perguntas
- Visualizações com Matplotlib e Seaborn

## 💡 Insights principais

- França foi a mais dominante — 10 gols e só 2 sofridos
- Brasil teve a melhor defesa do top 10 — apenas 1 gol sofrido
- Posse de bola tem correlação moderada com gols (0.55) — não é fator decisivo
- Haiti foi a mais agressiva — 55 faltas cometidas
- Argentina, Espanha e México não sofreram nenhum gol
